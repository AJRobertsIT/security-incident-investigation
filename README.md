# Security Incident Investigation

## Project Overview

This project documents the investigation of a simulated cybersecurity incident involving a compromised web server. The investigation focused on identifying malicious activity, analyzing HTTP traffic, determining the root cause of the incident, and recommending security improvements.

This project was completed as part of the Google Cybersecurity Professional Certificate and demonstrates practical incident response and security analysis skills.

## Investigation Summary

The investigation examined suspicious network activity affecting a company web server. Analysis included reviewing packet captures, identifying malicious connections, examining HTTP requests, determining how attackers gained access, and documenting recommended remediation steps.

Security recommendations included improving authentication controls, strengthening password policies, implementing Multi-Factor Authentication (MFA), and improving monitoring capabilities.
