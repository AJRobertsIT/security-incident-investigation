# Security Incident Investigation

## Project Overview

This repository documents a simulated cybersecurity incident investigation completed as part of the Google Cybersecurity Professional Certificate.

The project demonstrates the process of identifying suspicious activity, analyzing network traffic, determining the root cause of a security incident, and developing recommendations to improve an organization's security posture.

---

## Objectives

- Investigate suspicious network activity
- Analyze HTTP traffic and network communications
- Identify indicators of compromise (IOCs)
- Document findings in a professional incident report
- Recommend security improvements to reduce future risk

---

## Skills Demonstrated

- Incident Response
- Network Traffic Analysis
- HTTP Analysis
- Cybersecurity Documentation
- Security Monitoring
- Threat Identification
- Risk Analysis
- Technical Report Writing
